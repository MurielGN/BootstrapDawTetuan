Tipografía en Bootstrap 5



Configuración por defecto de Bootstrap 5

✨Font-size: 16px.

✨line-height: 1.5.

✨font-family : «Helvetica Neue», Helvetica, Arial, sans-serif.

✨Margin: Los elementos parrafo tienen margin-top: 0 y margin-bottom: 1rem

✨Encabezados: Del h1-h6. Los diseños de encabezados en Bootstrap son diferentes a los de HTML, siendo estos últimos ligeramente más pequeños.

Otros elementos de Bootstrap

mark
A este elemento de marcación, Bootstrap le añade un diseño tipo resaltado con rotulador.
Esto es una prueba de mark.

abbr
Este elemento que sirve para añadir significado a las abreviaciones. Bootstrap le añade un subrayado especial para que se vea que esconde ese texto explicativo.
DAW es un grado superior.

code
Esta etiqueta se usa para hacer referencia a códigos reservados. Bootstrap lo presenta con otro color.
Esto es un code

kbd
En HTML se usa para identificar comandos de teclado o formulas, se presentan con un color de fondo negro sobre letra en blanco.
Esto es kbd + Y

dl, dt, dd
Estas etiquetas son para definir definiciones de términos. Bootstrap resalta la palabra que luego se define.

Bootstrap
- Es un FrameWork.
small

En Boostrap se usa para crear un titular más pequeño debajo del encabezado.

SMALL Prueba small


blackquote

Esta etiqueta se usa para cuando introducimos un texto copiado de otra fuente. Bootstrap añade a la etiqueta footer un diseño tipo para señalar que la fuente es otro.

Este es un texto de prueba.

Otro tipo de fuente


Etiquetas de tipografía

✨.font-weight-bold, .font-weight-normal, .font-weight-light. Permite añadir grosor a la fuente.

✨.font-italic Italica. Añade cursiva.

✨.lead. Destaca el elemento con una fuente un poco más grande.

✨.small. Añade una fuente más pequeña.

✨.small. Etiquetas para alinear el texto dentro de un bloque.

✨.text-lowercase, .text-uppercase, .text-capitalize. Estos atributos permiten hacer que el texto sea todo minúsculas, mayúsculas, o que todas las palabras empiecen por mayúsculas.

✨.list-unstyled. Quita el list-style y margin-left en elementos de lista ul u ol.

✨.list-inline. Coloca los elementos li de una lista en una línea horizontal.

                      <dt>Bootstrap</dt>

                  </dl>

-Debido al uso generalizado de tablas en widgets de terceros, como calendarios y selectores de fechas, hemos diseñado nuestras tablas para que sean opcionales.

Simplemente agregue la clase base .table a cualquier table, luego amplíe con estilos personalizados o nuestras diversas clases de modificadores incluidas. Usando el marcado de tabla más básico, así es como se ven las tablas basadas en .table en Bootstrap. Todos los estilos de tabla se heredan en Bootstrap 4, lo que significa que cualquier tabla anidada tendrá el mismo estilo que la principal.

rounded float-left

rounded mx-auto d-block

Alinee las imágenes con las clases flotantes auxiliares o las clases de alineación de texto. las imágenes a nivel de bloque se pueden centrar utilizando la clase de utilidad de margen .mx-auto.

img-thumbnail

Además de nuestras utilidades de radio de borde, puede usar .img-thumbnail para dar a una imagen una apariencia de borde redondeado de 1 px.

Jumbotron


Jumbotron es una clase que facilita la creación de bloques con un fondo diferente, pensados para llamar la atención y muy útiles para crear cabeceras o separar contenido dentro de una estructura. Se añade la etiqueta jumbotron. Si se quiere que el texto dentro del bloque ocupe el 100% se añade jumbotron-fluid, además quita el redondeado de la clase jumbotron.

Ejemplos de Utilities y Flexbox en Bootstrap 5



Clases de bordes


Posición


Las clases de borde se utilizan para diseñar el borde y el radio del borde de elementos como imágenes, botones...

Código fuente


                  Borde izquierdo: <span class="border-start border-4"> </span>
                  Borde derecho: <span class="border-end border-4"> </span>
                  Tres bordes: <span class="border border-end-0 border-4"> </span>
                  Cuatro bordes: <span class="border rounded border-4"> </span>

Color


Las clases de color se utilizan para cambiar el color del borde de su página web y transmitir un significado específico. Son muy útiles a la hora de remarcar algo de importancia en nuestra web.

Código fuente


                  Borde verde: <span class="border border-success"> </span>
                  Borde rojo: <span class="border border-danger"> </span>
                  Borde amarillo: <span class="border border-warning"> </span>
                  Borde azúl: <span class="border border-info"> </span>
                  Borde suave: <span class="border border-light"> </span>

Clases de alineación del texto


Las clases de alineación de texto se utilizan para alinear el texto a diferentes posiciones de la pantalla.

Texto alineado a la izquierda en todos los tamaños.

Texto alineado en el centro en todos los tamaños.

Texto alineado a la derecha en todos los tamaños.

Texto que se ajusta al tamaño


Texto normal.

Texto fino.

Texto ancho.

Texto enlace que no muestra que es un enlace

                  <p class="text-end">Texto alineado a la derecha en todos los tamaños.</p>
                  <p class="badge bg-primary text-wrap" style="width: 6rem;">Texto que se ajusta al tamaño</p>

Clases de fondos


Las clases de fondos se utilizan para aplicar color al fondo del elemento. Tienen gran utilidad a la hora de notificar si el envío ha sido correcto o ha surgido un error.

Cargando...
Carga completa
ERROR

                  <div class="p-3 mb-2 bg-secondary text-white">Cargando...</div>
                  <div class="p-3 mb-2 bg-success text-white">Carga completa</div>
                  <div class="p-3 mb-2 bg-danger text-white">ERROR</div>

Flexbox, es una de las grandes diferencias entre Bootstrap 5 y Bootstrap 4, ya que este no incluía esta característica. Con flexbox, puedes administrar rápidamente el diseño, la alineación y el tamaño de la cuadrícula. Facilitan la creación de menús.
