PAGINACIÓN



Para crear una paginación básica, hay que agregar la clase pagination a un elemento "ul".
Luego agregar la clase page-item a cada elemento "li" y una clase page-link a cada enlace "a" dentro de "li".
La clase active se utiliza para "resaltar" la página actual.
La clase disabled se utiliza para enlaces en los que no se puede hacer click.
La clase pagination-lg sirve para hacer los bloques más grandes y pagination-sm para hacerlos más pequeños.
también podemos alinear los bloques en la izquierda, centro o derecha con:
class="pagination" para ponerlo a la izquierda, class="pagination justify-content-center" para centrar la paginación, y class="pagination justify-content-end" para situarlo a la derecha.
Otra forma de paginación son las migas de pan: Las clases breadcrumb y breadcrumb-item indican la ubicación de la página actual dentro de una jerarquía de navegación.

CARDS



Una card (tarjeta) es un contenedor de contenido flexible y extensible. Incluye opciones para encabezados y pies de página, una amplia variedad de contenido, colores de fondo contextuales y potentes opciones de visualización. Nos permite dar un estilo y una estructura a nuestro contenedor, se construyen con la menor cantidad de marcas y estilos posible, pero aun así logran ofrecer una tonelada de control y personalización. Para comenzar a utilizar la propiedad card de bootstrap, comenzamos con un div general con class="card". Y dentro ya podemos aplicar estilos con distintas clases de card.


  • card-header ,card-body, card-footer: nos indica que parte de la card es, cabezera, cuerpo o pie de página.
  • Títulos (se usan al agregarlos .card-title a una etiqueta h ), texto (.card-subtitle a una etiqueta h para agregar un subtítulo, y .card-title para agregar un título) y enlaces (agregar .card-link a una etiqueta)

NAVS



La navegación disponible en Bootstrap comparte marcas y estilos generales, desde la clase base nav hasta los estados activo y deshabilitado. El componente base nav está construido con flexbox y proporciona una base sólida para construir todo tipo de componentes de navegación. Incluye algunas anulaciones de estilo (para trabajar con listas), algo de relleno de enlaces para áreas de acceso más grandes y estilo básico deshabilitado.

NAV BARS



Una barra de navegación es un encabezado de navegación que se coloca en la parte superior de la página. Con Bootstrap, una barra de navegación puede extenderse o contraerse, según el tamaño de la pantalla.

Esto es lo que necesita saber antes de comenzar con la barra de navegación:

  • Las barras de navegación requieren un envoltorio para las clases.
  • Las barras de navegación y su contenido son fluidos de forma predeterminada. Cambie el contenedor para limitar su ancho horizontal de diferentes maneras.
  • Las barras de navegación responden de manera predeterminada, pero puede modificarlas fácilmente para cambiar eso.

LOGO

Navegación con contenido externo

Contenido Colapsado
Conmutable a través de la marca de la barra de navegación.

A veces, desea utilizar el complemento de colapso para activar un elemento contenedor para el contenido que se encuentra estructuralmente fuera del archivo .navbar. ¡ Debido a que nuestro complemento funciona en idy data-bs-targetemparejando, eso se hace fácilmente!

<div class="collapse" id="navbarToggleExternalContent">
   <div class="bg-dark p-4">
     <h5 class="text-white h4">Contenido Colapsadot</h5>
     <span class="text-muted">texto a través del NAVBAR.</span>
   </div>
   </div>
    <nav class="navbar navbar-dark bg-dark">
   <div class="container-fluid">
    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarToggleExternalContent" aria-controls="navbarToggleExternalContent" aria-expanded="false" aria-label="Toggle navigation">
    <span class="navbar-toggler-icon"></span>
   </button>
  </div>
</nav>

Barra de Navegación con formularios

Navbar

Se puede colocar varios controles de formulario y componentes dentro de una barra de navegación:

Los elementos secundarios inmediatos de .navbar tienen un diseño flexible y se establecerán de forma predeterminada en justify-content: space-between. Use utilidades flexibles adicionales según sea necesario para ajustar este comportamiento.

<nav class="navbar navbar-light bg-light">
  <div class="container-fluid">
   <a class="navbar-brand">Navbar</a>
   <form class="d-flex">
   <input class="form-control me-2" type="search" placeholder="Search" aria-label="Search">
   <button class="btn btn-outline-success" type="submit">Search</button>
  </form>
</div>
</nav>

Barra de navegación fuera del lienzo Offcanvas

Transforme su barra de navegación que se expande y contrae en un cajón fuera del lienzo con el complemento fuera del lienzo. Extendemos los estilos predeterminados de offcanvas y usamos nuestras .navbar-expand-*clases para crear una barra lateral de navegación dinámica y flexible. En el siguiente ejemplo, para crear una barra de navegación fuera del lienzo que siempre esté colapsada en todos los puntos de interrupción, omita la .navbar-expand-*clase por completo.
